Instructions

Quick Start

  1. Pick an example or start from the default model.
  2. Draw boundaries with EP, FL, Phreatic, and Impermeable Area tools.
  3. Switch to Standpipe and click to read head and water rise.
  4. Tune Kx, Ky, grid size, and EP/FL interior counts in Settings.

Tool Meanings

  • EP: fixed hydraulic head line.
  • Phreatic: fixed head equal to elevation.
  • FL: impermeable no-flow line.
  • Impermeable Area: no-flow polygon.
  • Select: move/edit existing boundaries and polygons.

View + Editing Controls

  • Wheel/pinch to zoom.
  • Drag empty canvas space (in Select) to pan.
  • Fit resets to full-domain view.
  • h toggles head-value shading overlay.
  • In Select: Alt+edge click adds a vertex, Ctrl/Cmd+vertex click removes one.
  • Esc cancels drawing. Delete removes the selected item.

Save / Load / Export

  • Save PNG exports the current canvas image.
  • Save state exports model setup (domain, solver, boundaries, view) as JSON.
  • Load state by dragging a saved JSON or `.flownet2` file onto the app.

Notes

  • Outer boundary is no-flow by default unless overridden by user-drawn EP/phreatic lines.
  • Coordinates mode switches between real (x, y) and transformed (x', y') space.
